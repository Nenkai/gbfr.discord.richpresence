# gbfr.discord.richpresence

A separate `social-sdk` branch exists with an attempt at interfacing the Discord Social SDK in C# (at least what is needed for using Rich Presence).

Mainly because discord-csharp-rpc seems to have issues with anything related to party invites/joins.
